Meet App

"Meet App" is a React application that helps users find events by city and select the number of events to display. It features city search, event filtering, and data visualizations with charts. The app also includes alerts for offline status and notifications for a better user experience.

Technologies used:

  • React
  • Google Calendar API
  • AWS Lambda
  • Serverless
  • TDD
  • PWA features
